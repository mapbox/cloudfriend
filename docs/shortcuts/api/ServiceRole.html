Class: ServiceRole

ServiceRole(options)

Create an IAM role that will be assumed by an AWS service, e.g. Lambda or ECS.

Constructor

new ServiceRole(options)

Parameters:
Name Type Description
options Object Extends the options for [`Role`](#role). You do not need to provide an `AssumeRolePrincipals` attribute, but do need to include the following additional attributes:
Properties
Name Type Description
Service String The name of the AWS service that will assume this role, e.g. `lambda`.

Example
const cf = require('@mapbox/cloudfriend');

const myTemplate = { ... };

const role = new cf.shortcuts.ServiceRole({
  LogicalName: 'MyRole',
  Service: 'lambda',
  Statement: [
    {
      Effect: 'Allow',
      Action: 's3:GetObject',
      Resource: 'arn:aws:s3:::my-bucket/my/data.tar.gz'
    }
  ]
});

module.exports = cf.merge(myTemplate, role);
