Class: GlueTable

GlueTable(options)

Create a Glue Table. Pre-configured versions of this shortcut are available for tables stored as line-delimited JSON or ORC: - [GlueJsonTable](#gluejsontable) - [GlueOrcTable](#glueorctable)

Example
const cf = require('@mapbox/cloudfriend');

const myTemplate = { ... };

const table = new cf.shortcuts.GlueTable({
  LogicalName: 'MyTable',
  DatabaseName: 'my_database',
  Name: 'my_table',
  Columns: [
    { Name: 'column_name', Type: 'string', Comment: 'my_column description' }
  ]
});

module.exports = cf.merge(myTemplate, table);
